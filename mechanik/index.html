Mechanik

Die Mechanik ist ein Teilgebiet der Physik, das sich mit der Bewegung von Körpern und den dabei auftretenden Kräften beschäftigt.

# Gleichförmige Bewegung

Die gleichförmige Bewegung ist eine Bewegung, bei der sich ein Körper mit konstanter Geschwindigkeit in eine Richtung bewegt. Dies bedeutet, dass der Körper in gleichen Zeitabständen gleiche Strecken zurücklegt. Ein Beispiel für eine gleichförmige Bewegung ist ein Auto, das mit einer konstanten Geschwindigkeit auf einer geraden Straße fährt. In der Physik wird die gleichförmige Bewegung oft durch die Formel \(s = v \cdot t\) beschrieben, wobei \(s\) die zurückgelegte Strecke, \(v\) die konstante Geschwindigkeit und \(t\) die Zeit ist.

# Beschleunigte Bewegung

Die beschleunigte Bewegung ist eine Bewegung, bei der sich die Geschwindigkeit eines Körpers ändert. Dies kann entweder eine Zunahme oder eine Abnahme der Geschwindigkeit sein. Ein Beispiel für eine beschleunigte Bewegung ist ein Auto, das von einer Ampel anfährt und dabei immer schneller wird. In der Physik wird die beschleunigte Bewegung oft durch die Formel \(a = \frac{\Delta v}{\Delta t}\) beschrieben, wobei \(a\) die Beschleunigung, \(\Delta v\) die Geschwindigkeitsänderung und \(\Delta t\) die Zeit ist, in der die Änderung stattfindet. Beschleunigte Bewegungen können auch durch Kräfte verursacht werden, wie zum Beispiel die Schwerkraft, die einen fallenden Apfel beschleunigt.
